### About Dulist

Dulist is a task management application designed to streamline productivity. With Dulist, users can efficiently manage their tasks and deadlines using an intuitive interface. It leverages powerful tools and frameworks to provide a responsive and robust experience.

### Key Features

- **Task Management**: Add, edit, and remove tasks effortlessly.
- **Due Dates**: Assign deadlines to tasks to stay organized.
- **Modern Design**: Built with Material UI for a polished and professional look.
- **State Management**: Powered by Redux for consistent and scalable state handling.
- **Type Safety**: Written in TypeScript to ensure robust code quality.

---

### Technologies Used

- **Vite.js**: For fast and efficient development.
- **Material UI**: To create a sleek and user-friendly design.
- **Redux**: For managing application state.
- **TypeScript**: To enhance maintainability and reduce runtime errors.
