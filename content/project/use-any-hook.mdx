### About `use-any-hook`

`use-any-hook` is a library designed to streamline state management and common functionality in React applications. It eliminates repetitive boilerplate, allowing developers to focus on building intuitive, scalable UIs.

React hooks have revolutionized the way we manage state and logic in functional components. However, certain scenarios like fetching data or toggling state often require writing similar boilerplate code repeatedly.

With `use-any-hook`, you get a suite of battle-tested hooks that save you time and effort while keeping your codebase clean and consistent.

### Getting Started

To install the library, run the following command:

```bash
npm i use-any-hook
```

Start by importing and using one of the hooks in your React project:

```jsx
import { useFetch } from "use-any-hook";
function MyComponent() {
  const [data, loading, error] = useFetch("https://api.example.com/data");

  return (
    <div>
      {loading ? "Loading..." : null}
      {error ? `Error: ${error.message}` : null}
      {data ? <pre>{JSON.stringify(data, null, 2)}</pre> : null}
    </div>
  );
}
```

### Built-in Hooks

Here are some of the key hooks included in `use-any-hook`:

#### **1. useFetch**

Fetch and handle data with ease.

```jsx
const [data, loading, error] = useFetch("https://api.example.com/resource");
```

#### **2. useToggle**

Effortlessly toggle between true and false states.

```jsx
const [isOn, toggle, setIsOn] = useToggle();

<button onClick={toggle}>Toggle</button>
<button onClick={() => setIsOn(true)}>Turn ON</button>
<button onClick={() => setIsOn(false)}>Turn OFF</button>
```

#### **3. More Hooks**

Want to explore more hooks like `useDebounce`? Check out the full list of hooks and their examples on the official [use-any-hook npm page](https://www.npmjs.com/package/@saif.dev/use-any-hook).
